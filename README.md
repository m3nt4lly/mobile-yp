# NATK Schedule Mobile App

## Описание
Мобильное приложение для просмотра расписания Новосибирского авиационного технического колледжа имени Б.С. Галущака.
Приложение разработано с использованием **Jetpack Compose** и следует принципам **Material Design 3**.

## Функционал
- **Просмотр расписания:** Отображение занятий на ближайшие 30 дней.
- **Поиск групп:** Удобный поиск с фильтрацией по названию группы.
- **Избранное:** Возможность сохранять группы в избранное для быстрого доступа.
- **Оффлайн режим:** (Частично) Сохранение последних просмотренных данных (планируется).
- **Темная тема:** Поддержка системной темной темы.

## Технологии
- Kotlin
- Jetpack Compose
- Retrofit (Network)
- Coroutines (Async)
- SharedPreferences (Persistence)

## Сборка и запуск
1. Откройте проект в Android Studio.
2. Убедитесь, что сервер (Backend) запущен на `localhost:5000` (для эмулятора доступен как `10.0.2.2`).
3. Запустите приложение на эмуляторе или реальном устройстве.
